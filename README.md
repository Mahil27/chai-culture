# Chai Culture - Landing Page

## How to Run

### Option 1: Quick View
Simply open the `index.html` file in any web browser.
- Double-click `index.html` in your file explorer.
- Or drag and drop `index.html` into a browser window.

### Option 2: Local Server (Recommended)
For the best experience (especially for avoiding CORS issues if we add more complex assets later), run with a local server.

If you have Node.js installed:
1. Open a terminal in this directory:
   ```powershell
   cd d:\chai
   ```
2. Run the server:
   ```powershell
   npx http-server
   ```
3. Open the URL shown (usually `http://127.0.0.1:8080`).

## Project Structure
- `index.html` - Main landing page.
- `style.css` - Styles and animations.
- `script.js` - Countdown timer and form logic.
- `logo.png` - Brand asset.
